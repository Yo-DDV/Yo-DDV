<details>
  <summary>How these metrics are produced</summary>

These cards use GitHub's public GraphQL and REST APIs. Activity is measured over the previous twelve months. The language breakdown comes from changed lines in merged public pull requests; documentation, lockfiles, vendored code and generated files are excluded. It reflects public contribution history, not a claim of professional proficiency in every language shown.

The repository generates its own SVG files with the scoped `GITHUB_TOKEN`. It does not rely on a personal token or an external statistics service.
